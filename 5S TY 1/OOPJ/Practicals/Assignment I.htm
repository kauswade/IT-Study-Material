Assignment I

 

  1. Implement a class Student. Student has a name and a total quiz score. Supply an appropriate constructor and methods getName(), addQuiz(int score), getTotalScore(), and getAverageScore(). To compute the latter, you also need to store the number of quizzes that the student took. Supply a StudentTester class that tests all methods.

 

  1. Implement a class Product. A product has a name and a price, for example new Product(“Toaster”, 29.95). Supply methods getName(), getPrice(), and reducePrice(). Supply a program ProductPrinter that makes two products, prints the name and price, reduces their prices by Rs. 5.00, and then prints the prices again.

 

  1. Write a class SavingsAccount that can perform the following operations
    1. Deposit money
    2. Withdraw money
    3. Get the current balance
    4. Get the simple interest
    5. Get the compound interest
    6. Get the balance after adding the interest

Provide suitable methods to perform above operations. Provide suitable constructors to construct objects of the class. Write a SavingsAccountTester class that constructs a savings account with an initial balance of Rs 10000 and a simple interest rate of 3% and compound interest rate of 10%. Print all the results after finding the simple interest and compound interest.

 

  1. Write static methods
    1. public static double sphereVolume(double r)
    2. public static double sphereSurface(double r)
    3. public static double cylinderVolume(double r, double h)
    4. public static double cylinderSurface(double r, double h)
    5. public static double coneVolume(double r, double h)
    6. public static double coneSurface(double r, double h)

that compute the volume and surface area of a sphere with radius r, a cylinder with circular base with radius r and height h, and a cone with circular base with radius r and height h. Place them into a class Geometry. Then write a program that prompts the user for the values of r and h, calls the six methods, and prints the results.
