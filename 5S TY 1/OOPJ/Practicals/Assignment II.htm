  1. Create an inheritance hierarchy for classes Quadrilateral, Trapezoid, Parallelogram, Rectangle and Square. Use Quadrilateral as base class of the hierarchy. Specify the instance variables and methods for each class. The private instance variables of Quadrilateral should be the x-y coordinate pairs for the four end points of the Quadrilateral. Write a test program that instantiates objects of your classes and outputs each object’s area (except Quadrilateral).

 

  1. Make a class Employee with a name and salary. Make a class Manager inherit from Employee. Add an instance field, named, department, of type String. Supply a method toString that prints the manager’s name, department, and salary. Make a class Executive inherit from Manager. Supply appropriate toString methods for all classes. Supply a test program that tests these classes and methods.

 

  1. Modify the SavingsAccount class of Assignment I Q. 3. It should throw IllegalArgumentException when the account is constructed with a negative balance, when a negative amount is deposited, or when an amount that is not between 0 and the current balance is withdrawn. Write a test program that causes all three exceptions to occur and that catches them all.
